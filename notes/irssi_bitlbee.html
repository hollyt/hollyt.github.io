Setting up irssi + bitlbee

I want to use screen + irssi + bitlbee as my perfect, customizable, IM client, but I had a bit of trouble getting everything to work correctly. This is mostly a set of notes to remind me how to get everything set up, but it might be helpful to you too.

Install the programs

I use Debian, and eveything we need is included in the package manager. You’ll need to obtain gnu screen, irssi, and bitlbee.
sudo apt-get install screen irssi bitlbee

I already used screen, so I’m not going to go into detail about how to set that up. For more information on integrating screen with irssi and bitlbee look here.

Configure bitlbee

Before launching irssi + bitlbee, you’ll want to configure bitlbee. Open up /etc/bitlbee/bitlbee.conf and follow the directions in the documentation to configure your bitlbee setup. Uncomment a line to implement/change that config. I changed:
RunMode = ForkDaemon
So bitlbee will run as a daemon.
User = bitlbee
So bitlbee will run as its own user - NOT root.
DaemonInterface = 0.0.0.0 DaemonPort = 6667
Uncomment the port info if you’re running as a daemon …
Operpassword = <password>
Set the oper password …

The rest of the config file is commented well so just read through it and make any changes as you see fit.

Using irssi + bitlbee

Enter irssi by typing irssi in the command line. To connect to the bitlbee server: /server localhost You should be in the room &bitlbee (the bitlbee control channel) and see two users in the chat - yourself and root. You can interact with root to set up and interface with bitlbee.

NOTE: irssi commands are prefixed with / - bitlbee commands are not.

First, make sure that you’re using a nickname you’re happy with. To change your nickname in irssi: /nick <nickname>
Now register a password in the bitlbee control channel: register <password>
Next time you return to the bitlbee control channel, use identify <password> to identify yourself. This will load all of the irc & im accounts associated with your bitlbee user.
For example:
/nick my_nickname
register mypw123

Add IM accounts to bitlbee

While in your bitlbee account, type the following to add im accounts:
account add <protocol> <username>
/OPER
Now enter your IM password.

For example:
account add jabber my_jabber_name@myxmppserver.net
/OPER

To list the accounts stored in bitlbee:
account list

To connect to an account: account <account #> on

You can always use help account if you get suck.

Chatting

After you have turned your account on, you should see any buddies who are currently online join the room. To chat with a buddy, address them directly like this:
my_buddy's_name: hello
Or message them privately with /m <buddy_name> <msg> and /query <buddy_name>.

Notes

Once you get your initial setup configured and you’ve added your IM accounts, you can make your server private to only registered users by changing the following in bitlbee.conf:
AuthMode = Registered
So only registered users who know your password can use your server.
Authpassword = <password>
